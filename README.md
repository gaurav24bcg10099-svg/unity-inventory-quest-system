#  Unity Inventory & Quest System

##  Overview
A Unity-based system that allows players to manage inventory, interact with NPCs, and complete quests.

---

##  Features
- Inventory system with slots
- Drag & drop functionality
- Hotbar system
- NPC interaction (Press E)
- Quest system (Collect items)

---

##  Tech Stack
- Unity Engine
- C#
- Unity UI (Canvas)

---

## Controls
- **E** → Interact with NPC

---
